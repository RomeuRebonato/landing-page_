Matérias atuais da faculdade


Atualmente estou no primeiro período do curso Sistema de Informação na Universidade de Vila Velha e essas são as matérias atuais que estou cursando

Matérias Sobre a matéria
Construção de Software para Web Tem como objetivo ensinar o desenvolvimento de aplicações web, abordando desde os conceitos fundamentais até práticas modernas de desenvolvimento.
Design e Desenvolvimento de Banco de Dados I Ensina os fundamentos do armazenamento, organização, recuperação e manipulação de dados em sistemas computacionais. Ela é essencial para o desenvolvimento de sistemas de informação, aplicativos e softwares que precisam lidar com grandes volumes de dados.
Experiência e Interface com o Usuário Essa disciplina é voltada para o design e desenvolvimento de interfaces digitais focadas nas pessoas, buscando tornar os sistemas mais intuitivos, acessíveis e agradáveis de usar.
Fundamentos de Tecnologia da Computação É a introdução ao universo da computação, abordando os principais conceitos, componentes e funcionamento dos sistemas computacionais.
Lógica para Computação base essencial para o raciocínio lógico e para a programação, muito usada no desenvolvimento de algoritmos, linguagens formais e sistemas computacionais em geral.
Textos Científicos: Aspectos Metodológicos e Linguísticos Aborda a produção, estrutura e linguagem dos textos acadêmicos e científicos, fundamentais no ambiente universitário.
